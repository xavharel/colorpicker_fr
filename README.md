# HTML Color Reference
(Le français suit)

Static one‑page reference for CSS/HTML named colors, showing:
- a visual swatch for each color,
- English names and suggested French names,
- Hex, RGB and HSL values,
- click‑to‑copy for Hex codes.

## Usage

1. Download or clone this repository.
2. Open `index.html` in your browser (double‑click or drag‑and‑drop into a browser window).
3. Browse the sections by color family (reds, greens, blues, etc.).
4. Click any Hex code to copy it, then paste it into your editor or design tool.

## Development

This is a self‑contained static HTML page:
- no build step or external dependencies,
- all HTML, CSS and minimal JavaScript live in `index.html`.

You can freely edit `index.html` to:
- tweak the French names,
- add or remove colors,
- change the visual style.

## Référence de couleurs HTML

Petit outil statique pour parcourir les couleurs nommées CSS/HTML, avec :
- un aperçu visuel,
- les noms anglais et des propositions de noms français,
- les valeurs Hex, RGB et HSL,
- un clic sur le code Hex pour le copier dans le presse‑papiers.

### Utilisation

1. Téléchargez ou clonez ce dépôt.
2. Ouvrez le fichier `index.html` dans votre navigateur (double‑clic ou glisser‑déposer dans une fenêtre de navigateur).
3. Parcourez les sections par famille de couleurs (rouges, verts, bleus, etc.).
4. Cliquez sur un code Hex pour le copier, puis collez‑le dans votre éditeur ou votre outil de design.

### Développement

Ce projet est une page HTML autonome:
- aucun build ni dépendance externe,
- tout le code (HTML, CSS et JavaScript minimal) est dans `index.html`.

Vous pouvez modifier librement `index.html` pour:
- ajuster les noms français,
- ajouter/retirer des couleurs,
- adapter le style de la page.
